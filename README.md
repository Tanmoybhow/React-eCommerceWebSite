# 🛍️ VoltMart – Modern E-Commerce Web Application

VoltMart is a modern, responsive e-commerce web application built with **React**. It provides a seamless shopping experience with product browsing, authentication, cart management, filtering, dark mode, and location detection.

## ✨ Features

- 🛒 Browse products from Fake Store API
- 🔍 Search and filter products
- 📂 Filter by category
- 🏷️ Filter by brand
- 💰 Filter by price range
- 📱 Fully responsive design
- 🌙 Light & Dark Mode
- 🔐 User Authentication using Clerk
- 👤 User Profile with Clerk
- 🛍️ Shopping Cart
- 💾 Cart persistence using Redux Persist
- 📍 Detect user's live location using Geolocation API
- 📧 Contact form powered by Formspree
- ⚡ Fast navigation using React Router
- 🔄 State management using Redux Toolkit
- 💾 Theme preference stored in Local Storage
- 🎨 Modern UI built with Tailwind CSS
- 📦 Pagination for products

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/voltmart.git
```

Navigate to project

```bash
cd voltmart
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file.

```env
VITE_CLERK_PUBLISHABLE_KEY=YOUR_CLERK_KEY

VITE_OPENCAGE_API_KEY=YOUR_OPENCAGE_KEY
```

## 🌟 Key Functionalities

### Authentication

- Secure sign in and sign up with Clerk
- User profile management

### Shopping Cart

- Add products to cart
- Increase/Decrease quantity
- Remove products
- Persistent cart using Redux Persist

### Product Filtering

- Search products
- Category filtering
- Brand filtering
- Price filtering

### Dark Mode

- Toggle between light and dark mode
- Theme saved in Local Storage

### Location Detection

- Detect current location using Geolocation API
- Convert latitude and longitude into readable address using OpenCage API

### Contact

- Contact form integrated with Formspree
- Users can directly send messages
